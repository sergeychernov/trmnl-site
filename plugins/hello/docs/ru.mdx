# Приветствие

Плагин **«Приветствие»** показывает на экране TRMNL персональное приветствие и текущее время для выбранного часового пояса.  
Он идеально подходит как базовый экран для домашнего или рабочего устройства, где важно сразу увидеть, **который час** и **для кого настроено устройство**.

## Что делает плагин

- **Персонализированное приветствие**  
  Использует имя пользователя из общих настроек (`name`), чтобы выводить понятное человеку обращение.
- **Текущее время**  
  Отображает время для выбранного часового пояса (поле `timeZone` в настройках плагина).
- **Адаптация под размер экрана**  
  Макет рендерится как React‑компонент и корректно подстраивается под любые размеры блока.

## Основные настройки

- **Часовой пояс (`timeZone`)**  
  - Формат IANA, например: `Europe/Moscow`, `Europe/Berlin`, `America/New_York`.  
  - Определяет, какое локальное время будет показано пользователю.
- **Имя пользователя**  
  - Берётся из общих пользовательских настроек TRMNL (`name`).  
  - Если имя не заполнено, приветствие выводится в более нейтральном виде.

## Типичные сценарии использования

- Домашний TRMNL на входе в квартиру — короткое приветствие и текущее время.
- Рабочее место — напоминание о времени и принадлежности устройства конкретному человеку.
- Экспериментальный/демо‑экран для первых тестов BYOS‑устройства.

## Ограничения и рекомендации

- Плагин **не ходит во внешние API** и не использует данные из БД — только настройки пользователя и плагина.
- Для корректного отображения времени важно указать **правильный IANA‑часовой пояс**.
