# Telegram

Плагин **«Telegram»** превращает ваш TRMNL в персональный дисплей для сообщений из Telegram. Отправляйте текст через бота, и он мгновенно появится на экране устройства.

## Что делает плагин

- **Отображает сообщения из Telegram**  
  Показывает последнее сообщение, отправленное через бота TRMNL на это устройство.
- **Поддерживает форматирование Markdown**  
  Жирный, курсив, подчёркнутый, зачёркнутый текст, inline-код, ссылки и списки.
- **Настраиваемый размер шрифта**  
  5 уровней размера текста от «очень мелкого» до «очень крупного».
- **Отметка времени**  
  Внизу экрана показывается дата и время последнего сообщения.

## Как начать использовать

1. **Привяжите Telegram-аккаунт**  
   - Перейдите в профиль пользователя на сайте TRMNL.
   - Нажмите «Привязать Telegram» и следуйте инструкциям.
   - Отправьте боту код привязки командой `/start <код>`.

2. **Выберите плагин Telegram**  
   - В настройках устройства выберите плагин «Telegram».
   - Настройте размер шрифта по вкусу.

3. **Отправляйте сообщения**  
   - Напишите боту любой текст — он появится на экране.
   - Редактируйте сообщения — экран обновится автоматически.

## Команды бота

- `/start <код>` — привязать аккаунт к TRMNL
- `/devices` — список ваших устройств с плагином Telegram
- `/help` — справка по командам

## Поддержка форматирования

Плагин корректно отображает Telegram-форматирование:
- **Жирный текст** — `**текст**`
- *Курсив* — `*текст*`
- __Подчёркнутый__ — `__текст__`
- ~~Зачёркнутый~~ — `~~текст~~`
- `Моноширинный` — `` `код` ``
- [Ссылки](https://example.com) — `[текст](url)`

## Настройки

- **Размер шрифта (`fontScale`)**  
  - -2: Очень мелкий
  - -1: Мелкий  
  - 0: Обычный (по умолчанию)
  - +1: Крупный
  - +2: Очень крупный

## Типичные сценарии использования

- **Напоминания и заметки** — отправьте себе важную информацию через Telegram.
- **Семейный дисплей** — члены семьи могут оставлять сообщения друг другу.
- **Уведомления из других сервисов** — настройте переадресацию важных сообщений.
- **Мотивационные цитаты** — обновляйте вдохновляющий текст каждый день.

## Особенности работы

- Плагин использует стратегию `replace` — на экране всегда показывается только последнее сообщение.
- Если у вас несколько устройств с плагином Telegram, при отправлении сообщения выберите нужное.
- При редактировании сообщения в Telegram экран обновляется автоматически.
- Максимальная длина сообщения ограничена размером экрана и выбранным шрифтом.

## Безопасность

- Сообщения хранятся в защищённой базе данных TRMNL.
- Только вы можете отправлять сообщения на свои устройства.
- Привязка аккаунта происходит через уникальный одноразовый код.
